# Executive Headshot

Drop your professional photo here as `headshot.jpg` and the site will pick it
up automatically on the next build — no code changes needed. Until then, the
homepage and About page render a designed placeholder instead of a broken
image.

**Path:** `public/images/profile/headshot.jpg`

**Recommended photo specs:**
- 1200×1500px or larger, portrait orientation (4:5 ratio matches the layout best)
- Neutral background (soft gray, off-white, or a simple studio backdrop —
  avoid busy or outdoor backgrounds)
- Business or business-casual attire
- Professional, even lighting (soft key light, minimal harsh shadow)
- Sharp focus, high resolution — this photo is the credibility signal next to
  original frameworks and case studies, so it should read at consulting-firm
  quality, not a casual selfie

See the root `README.md` for the full explanation of how the placeholder ↔
real-photo swap works.
